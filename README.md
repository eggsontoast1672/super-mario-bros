# Super Mario Bros Clone

I've been bouncing around between some different languages and graphics
libraries recently, but I've had some sort of gripe with all of them. Rust
doesn't have a mature enough ecosystem, Python doesn't have static typing, Java
doesn't have any lightweight build tools available, C++ is too nonsensical, *et
cetera*. In light of that, I have decided to hunker down with C and see how
things go. Good old Simple Direct Media Layer has never done me any wrong.

This is quite a big project. I want to recreate a bunch of the levels from the
original game as faithfully as possible. That means copying the tiles, sound
effects, music, and game mechanics. Hopefully Nintendo does not find this and
try to take it down. Although if they do, I might consider that a win, since it
means I copied it accurately!
